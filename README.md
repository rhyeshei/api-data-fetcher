# 飲食店検索・予算フィルタリングツール

## プロジェクト概要
このプロジェクトは、ホットペッパーグルメWebサービス（API）を利用して、特定のキーワード（例：五反田駅）に関連する飲食店情報を取得し、予算に応じた絞り込みを行うデータ処理ツールです。

## 作成背景
本プロジェクトは、授業内で講師が作成したテキスト教材を参考に、Pythonを用いた外部APIとの連携および、Pandasによるデータ加工の学習の一環として作成されました。教材で学んだAPIリクエストの基本構造や、文字列操作によるデータクレンジングの手法を応用して実装しています。

## 使用技術
- **言語**: Python 3.9.6
- **主なライブラリ**:
  - `requests`: ホットペッパーグルメAPIへのリクエスト送信
  - `pandas`: 取得したデータの構造化・フィルタリング・整形
  - `json`: APIレスポンス（JSON形式）の解析

## プログラムの機能
1. **API連携**: ホットペッパーグルメWebサービスから、指定したキーワード（初期設定では「五反田駅」）に一致する店舗情報を最大100件取得します。
2. **データクレンジング**: APIから取得した「3001～4000円」のような文字列形式の予算データを数値化し、分析やフィルタリングが可能な状態に変換します。
3. **予算フィルタリング**: ユーザーが設定した上限予算（例：4,000円以下）に基づいて、対象となる店舗のみを抽出して表示します。
4. **詳細情報の抽出**: 店名、住所、キャパシティ、アクセス情報、営業時間、URLなどを一覧形式で保持します。

## ファイル構成
- `app5.ipynb`: APIリクエストからデータ処理、フィルタリングまでを行うJupyter Notebook。
- `README.md`: 本ドキュメント。

## 使い方
1. [リクルート Webサービス](https://webservice.recruit.co.jp/)からAPIキーを取得します。
2. `app5.ipynb` 内の `API_KEY` 変数にご自身のキーを入力してください。
3. ノートブックを実行すると、条件に合致した店舗情報のリストが表示されます。

## 注意事項
- 本プログラムの実行には有効なAPIキーが必要です。
- APIの利用にあたっては、リクルートWebサービスの利用規約を遵守してください。
